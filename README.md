# Learning Python

This repo is dedicated to my path of learning Python.

## Installation

To test out the files I have created during the process of learning Python, just clone this repo locally via the **Code** Button above this README.
You can either choose to clone it using HTTPS or download a ZIP archive.

> [!NOTE]  
> You need to be able to run Python code locally on your device. 
> Refer to the [official Python website](https://www.python.org) for more information about how you can download Python to your device.

## Usage

Open a command line interface and move into the directory you downloaded this repo to. 
Execute one of the .py files and enjoy!

This repo contains several files which have helped me learn Python. Here is an overview of it's files and what they do.
